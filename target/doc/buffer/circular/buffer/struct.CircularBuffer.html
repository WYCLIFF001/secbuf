Expand description

A circular (ring) buffer for efficient streaming data.

Memory is automatically securely zeroed on drop.

§Example

use buffer::prelude::*;

let mut ring = CircularBuffer::try_new(1024)?;
ring.write(b"Hello, ")?;
ring.write(b"World!")?;

let mut output = vec![0u8; ring.used()];
ring.read(&mut output)?;
assert_eq!(&output, b"Hello, World!");

pub fn read_ptrs(&self) -> (&[u8], &[u8])

Returns slices for reading data (zero-copy).

Due to the circular nature, data may wrap around, requiring two slices. The second slice will be empty if data is contiguous.

pub fn write(&mut self, data: &[u8]) -> Result<usize>

Writes data into the circular buffer.

§Improved Implementation

This uses a simpler two-part copy approach instead of complex loop logic:

  1. Calculate how much can be written before wrap-around
  2. Write first part
  3. Write second part (if wrapped)
§Errors

Returns BufferError::InsufficientSpace if data doesn’t fit.
